# Nodos

Node.js framework for humans

* [site (under development)](https://nodosjs.github.io)
* [api documentation](https://nodosjs.github.io/nodos/)

## Requirements

* Nodejs >= 14

## Install

```sh
npm i -g @nodosjs/cli
nodos new MyBlog
cd MyBlog
npm install
nodos server # see http://localhost:8080/
# commands: nodos --help
```

## Development

We use https://lerna.js.org

```sh
# Setup
git clone <this repo>
cd nodos
make setup
make test

# test single package
npx jest packages/nodes-routing
```

### Example project


```sh
cd example
make test
make start
```
